# web-daw
간단한 교육용 웹 DAW를 만듭니다.

목표
1. 모든 구성을 4마디 기준으로 작업.
2. 8마디를 기준으로 하나의 파트로 구분.
3. Verse, Prechorus, Chorus의 3개의 파트 24마디를 구성, 24마디짜리 곡을 만드는 것을 최종 목표로함.
4. 간단한 피아노 반주와 함께 드럼, 베이스에 대해 이해.
5. 최종적으로 음악의 3요소인 멜로디, 화성, 리듬이 완성되는 과정을 이해시킴.

1. 각 노트 구현 완료
2. 코드로 넣는 것 구현 완료
3. 파트 구현 완료
4. 미디 출력 완료
